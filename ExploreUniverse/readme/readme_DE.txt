Das Schiff auf dem dieses Kommando ausgeführt wird, erkundet selbstständig unbekannte Sektoren. Dazu muss mindestens ein unerforschter Sektor an den aktuellen Sektor angrenzen. Anschließend wird jeweils zufällig ein dem Spieler unbekannter Sektor angeflogen. Sobald das Schiff sich in einem Sektor befindet an den keine unbekannten Sektoren mehr angrenzen beendet es seine Tour und sendet Anzahl und Namen der entdeckten Sektoren zurück.
